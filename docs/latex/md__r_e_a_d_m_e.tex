A command-\/line tool for managing personal finances. This project provides functionalities to add, modify, delete transactions, view logs, generate reports, and visualize financial data. It uses Python with pandas for data manipulation, matplotlib for plotting, and includes a logging system to track changes and errors.

When trying out the application, you can delete the CSV files or keep it.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md__r_e_a_d_m_e_autotoc_md1}

\begin{DoxyItemize}
\item {\bfseries{Add Transactions}}\+: Easily add new transactions with details including date, amount, category, and description.
\item {\bfseries{Modify Transactions}}\+: Update existing transactions with new details.
\item {\bfseries{Delete Transactions}}\+: Remove transactions from the records.
\item {\bfseries{View Logs}}\+: Access logs for transactions, new entries, updates, and deletions.
\item {\bfseries{Summary Reporting}}\+: View a summary of income and expenses, including total amounts and average transactions.
\item {\bfseries{Income/\+Expense Breakdown}}\+: Generate detailed reports of income and expenses by description.
\item {\bfseries{Visualization}}\+: Plot graphs to visualize income and expenses over time.
\item {\bfseries{Documentation}}\+: Documented with Doxygen for detailed code description.
